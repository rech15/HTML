HTML

É uma Linguagem de Marcação de Hipertexto baseada em tags criadas na década de 1990 por Tim Berners-Lee.

A HTML é ultilizado para publicação de conteúdo em hipertexto, tais como:
  • texto;
  • imagem;
  • áudio;
  • vídeo
    Este elementos podem ser vinculados através de hyperlinks.

    2014 - Versão HTML 5
    Aperfeiçoar a experiência da World Wide Web para os desenvolvedores e usuários finais

    TAGS

    As tags definem ao navegador a marcação de conteúdo.
    As tags tem funcionalidades espeíficas. Sem o use de tags o navegador exibe um texto plano.

    As tags podem possuir atributos, ou seja, são parametros que definem algum tipo de informação extra ao navegador.
